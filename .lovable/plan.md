## الهدف
1. في صفحة معاينة الطباعة (DocumentPreviewPage وما يشابهها): شريط "تخصيص الرؤية" البنفسجي يغطي الفاتورة على الجوال. اجعله مطوياً افتراضياً على الجوال مع زر لفتحه.
2. إخفاء التلميحات داخل قائمة اقتراحات البحث عن منتج في كل صفحات الإدخال:
   - "اكتب للبحث (X منتج)" (حالة hint قبل الكتابة)
   - "لا توجد نتائج" / "لم يتم العثور على المنتج" (حالة empty)
   هذا يطبَّق على كافة الصفحات والنُسخ.

## التغييرات

### 1) شريط تخصيص الرؤية قابل للطي على الجوال
- ملف: `src/components/PrintVisibilityToolbar.tsx`
- إضافة حالة `collapsed` افتراضية `true` عند `window.innerWidth ≤ 767`.
- عند الطي يظهر فقط زر صغير "تخصيص الرؤية" (eye icon). نقرة عليه تفتح الشريط الكامل.
- زر إغلاق (×) داخل اللوحة المفتوحة يطويها مجدداً.
- على الديسكتوب يبقى السلوك الحالي (مفتوح دائماً) دون تغيير.

### 2) إخفاء تلميحات اقتراحات البحث
- ملف عام: `src/index.css` — إضافة قاعدة CSS واحدة:
  ```css
  .search-suggestions .suggestions-status[data-status="hint"],
  .search-suggestions .suggestions-status[data-status="empty"] {
    display: none !important;
  }
  ```
- هذا يخفي العنصرين أينما ظهرت قائمة الاقتراحات (`SuggestionsPortal` + النسخ الداخلية للصفحات) بدون لمس كل صفحة على حدة.
- حالة "loading" تبقى ظاهرة (مفيدة للمستخدم).

## ما لن يتغير
- منطق البحث، نتائج المنتجات، أو أي بيانات.
- مظهر معاينة الطباعة نفسها.
- شريط الأدوات على الديسكتوب في كلا الميزتين.
